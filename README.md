# ai-maths-tutor
Interactive AI-based maths tutoring system with adaptive difficulty and step-by-step explanations

Features:
Adaptive difficulty (easy → medium → hard)
Step-by-step explanations for incorrect answers
Dynamic question generation across multiple topics
Reinforcement through similar question testing
Performance tracking and session summary
Chat-based UI for engaging learning interaction

Technologies Used:
React (JavaScript)
Custom logic for question generation and feedback

Purpose:
This project demonstrates the design of an intelligent tutoring system that simplifies complex mathematical concepts and adapts to user performance.
