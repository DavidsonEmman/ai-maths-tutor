# Smart Maths Tutor

Interactive AI-based maths tutoring system with adaptive difficulty and step-by-step explanations.

## Features
- Adaptive difficulty: Easy → Medium → Hard
- Step-by-step explanations for incorrect answers
- Dynamic question generation across multiple maths topics
- Reinforcement through similar question testing
- Performance tracking and session summary
- Chat-based user interface for engaging learner interaction

## Technologies Used
- React
- JavaScript
- Modular component structure
- Custom question generation and feedback logic

## Project Structure
- `components/` – UI components
- `constants/` – app phases and fixed values
- `data/` – topics and feedback messages
- `logic/` – question generation, answer checking, helpers, and explanations
- `styles/` – app styling

## Purpose
This project demonstrates the design of an intelligent tutoring system that simplifies mathematical concepts and adapts to user performance.
